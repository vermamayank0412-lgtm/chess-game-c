# ♟ Chess Game in C

A console-based 2-player Chess game built in C using **2D arrays, functions, and structures**.

## Features
- 8x8 Chess Board
- Piece Initialization
- Board Display
- Move Logic (in progress)
- Turn-based Gameplay (coming next)
- Check / Checkmate logic (future)

## Tech Used
- C Programming
- 2D Arrays
- Functions
- Header Files

## Project Goal
To build a fully functional chess game in C and strengthen understanding of **DSA, problem solving, and game logic**.
